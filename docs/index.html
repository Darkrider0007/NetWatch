NetWatch

NetWatch is an open-source Windows desktop application that displays active network connections together with process information, executable details, DNS resolution, GeoIP lookup, and connection history.

Download View Source
GPL-3.0 Windows 10 / 11 Python PySide6 SQLite

Overview

NetWatch focuses on showing which applications are communicating over the network. Each connection is enriched with process information, executable path, DNS hostname, remote country, connection statistics and historical records.

The application is intended for developers, system administrators, security researchers and users who want better visibility into network activity on Windows.

Features

Live Monitoring

Displays active TCP and UDP connections in real time.

Process Information

View process name, executable path, PID and publisher information.

DNS Resolution

Resolves remote IP addresses into hostnames whenever available.

GeoIP Lookup

Displays the country associated with remote IP addresses.

Connection History

Stores connection history locally using SQLite for later inspection.

Search & Filters

Quickly locate connections using process, host or IP address filters.

CSV Export

Export connection information for documentation or further analysis.

Desktop Notifications

Receive notifications when newly detected applications establish network connections.

Installation

  1. Download the latest release from the GitHub Releases page.
  2. Extract the downloaded ZIP archive.
  3. Open the extracted folder.
  4. Run NetWatch.exe.
  5. For the best experience, run the application with administrator privileges.

Getting Started

  1. Launch NetWatch.
  2. Wait a few seconds while active network connections are enumerated.
  3. Browse the connection table to view active TCP and UDP connections.
  4. Select a connection to view process information, executable path, hostname and country.
  5. Use the search box or filters to quickly locate a process, application or remote host.
  6. Export the current list of connections as a CSV file when required.

Microsoft SmartScreen

NetWatch is an open-source application and is currently distributed without a commercial code-signing certificate.

Because the application is not digitally signed, Windows SmartScreen may display a warning the first time it is launched.

This does not indicate that the application is malicious. The warning simply means that Windows cannot verify the publisher's identity.

  1. Click More info.
  2. Click Run anyway.

Since NetWatch is fully open source, you may inspect the source code, build the application yourself, or verify the release before running it.
